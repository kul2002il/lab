Введение

Цель

Формирование умений осуществлять процесс разработки с помощью CMS.

Задания для выполнения

  1. Придумайте и реализуйте структуру меню для вашего сайта (тематику можете выбрать сами). Добавьте не менее 5 статей.
  2. Установите и настройте плагин для изменения навигации между страницами (файл wp-pagenavi.2.92). Настройте и активируйте его.
  3. Разработайте плагин записи на техосмотр автомобиля.
    • Параметры формы: ФИО, Е-майл, Телефон, Марка автомобиля, Причина обращения.
    • Плагин реализуйте с помощью тегов шаблонов.
    • При отправке данные должны пройти валидацию (Е-майл должен быть уникальным, телефон не короче 11 символов, ФИО – кириллица с пробелами);
    • Отправленные заявки с формы должны быть доступны в административной панели для просмотра и удаления;
    • После успешной отправки заявки пользователю выдается сообщение.
  4. Разместите ваш сайт в сети интернет.
