Цель

Целью лабораторной работы является обучение методам и средствам сбора информации об анализируемом веб-приложении.

Задачи

Выполнить сбор информации об анализируемом веб-приложении https://www.php.net/

Ход работы

1. Проанализировано содержимое файла robots.txt.

User-agent: *
Disallow: /backend/
Disallow: /distributions/
Disallow: /stats/
Disallow: /server-status/
Disallow: /search.php
Disallow: /mod.php
Disallow: /manual/add-note.php
Disallow: /manual/vote-note.php

Disallow: /harming/humans
Disallow: /ignoring/human/orders
Disallow: /harm/to/self
Файл robots.txt

Файл состоит из указаний для роботов поисковых движков. Состоит из пар "Параметр: значение".

Судя по названию параметра "User-agent", он служит для указания всем агентам выполнять определённые инструкции. Т.к. данный параметр написан в самом начале, то он либо настраивает сразу весь документ, либо те инструкции, что указаны ниже него.

Инструкция "Disallow" служит для исключения из индексации поисковыми движками. Например "Disallow: /search.php". На данную страницу (https://www.php.net/search.php) Можно перейти и воспользоваться ей, однако информация о данной странице не доступна поисковикам:

Информация о странице недоступна поисковику

	<site-control permitted-cross-domain-policies="master-only"/>
